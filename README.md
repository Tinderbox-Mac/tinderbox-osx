## What is Tinderbox?  

**Tinderbox** is a powerful tool for organizing information, connecting ideas, and visualizing projects. It allows users to capture notes, link them, and explore complex relationships through interactive maps, outlines, and dashboards. Ideal for writers, researchers, and planners, Tinderbox transforms raw ideas into structured, searchable knowledge.  

With Tinderbox, every note can include text, images, tags, and custom attributes. Notes can be linked dynamically, mapped visually, or filtered using smart agents to reveal patterns and connections. This approach makes it easy to track projects, plan creative work, or organize research data efficiently.  

The app is optimized for **macOS**, supporting Retina displays and multiple windows. It integrates seamlessly with other tools, enabling users to combine content from different sources and visualize it in maps or outlines. Whether managing small personal projects or complex organizational data, Tinderbox provides a flexible and interactive workspace.
